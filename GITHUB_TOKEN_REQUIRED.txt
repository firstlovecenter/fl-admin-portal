┌─────────────────────────────────────────────────────────────┐
│  🚨 CRITICAL: GitHub Token Required for Amplify Build      │
└─────────────────────────────────────────────────────────────┘

WITHOUT THIS, YOUR BUILD WILL FAIL WITH:
├─ npm error 401 Unauthorized
└─ authentication token not provided

┌─────────────────────────────────────────────────────────────┐
│  📝 Quick Fix (3 steps, 2 minutes)                          │
└─────────────────────────────────────────────────────────────┘

1️⃣  CREATE GITHUB TOKEN
   ├─ Go to: https://github.com/settings/tokens
   ├─ Click: "Generate new token (classic)"
   ├─ Name: "AWS Amplify - FL Admin Portal"
   ├─ Scope: ✅ read:packages (ONLY THIS)
   ├─ Expiration: 1 year or No expiration
   └─ Click: "Generate token"
   
   ⚠️  COPY IT NOW! (starts with ghp_)

2️⃣  ADD TO AMPLIFY
   ├─ AWS Amplify Console → Your App
   ├─ App Settings → Environment variables
   ├─ Add variable:
   │  ├─ Name: GITHUB_TOKEN
   │  └─ Value: ghp_xxxxxxxxxxxxxxxxxxxxx
   └─ Click: Save

3️⃣  REDEPLOY
   ├─ Go to your app
   └─ Click: "Run build"

│  ✅ Verification                                            │
